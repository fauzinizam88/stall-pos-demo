# Stall POS Demo

Mobile-first static user-interface prototype for a small food-stall self-ordering and administration system.

## Public demo

After GitHub Pages is enabled for the `main` branch and `/ (root)`, the site is available at:

`https://fauzinizam88.github.io/stall-pos-demo/`

## Enable GitHub Pages

1. Open **Settings** in this repository.
2. Select **Pages** under **Code and automation**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Select branch **main** and folder **/ (root)**.
5. Select **Save**.

## Prototype scope

This repository currently contains a front-end demonstration only. It does not persist orders, verify payments, authenticate administrators, connect to Firestore, or deploy a Cloud Run backend.
